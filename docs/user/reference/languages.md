# 语言

RustCodeGraph解析的每种语言及其可识别的扩展。

从文件扩展名开始，语言支持是自动的—没有什么可配置的。

| 语言 | 插件 | 状态 |
|---|---|---|
| TypeScript | `.ts`、`.tsx` | 全力支持 |
| JavaScript | `.js`、`.jsx`、`.mjs` | 全力支持 |
| Python | `.py` | 全力支持 |
| Go | `.go` | 全力支持 |
| Rust | `.rs` | 全力支持 |
| Java | `.java` | 全力支持 |
| C# | `.cs` | 全力支持 |
| PHP | `.php` | 全力支持 |
| Ruby | `.rb` | 全力支持 |
| C | `.c`、`.h` | 全力支持 |
| C++ | `.cpp`、`.hpp`、`.cc` | 全力支持 |
| Swift | `.swift` | 全力支持 |
| Kotlin | `.kt`、`.kts` | 全力支持 |
| Scala | `.scala`、`.sc` | 完全支持（类、特征、方法、类型别名、Scala 3枚举） |
| Dart | `.dart` | 全力支持 |
| Svelte | `.svelte` | 全面支持（脚本提取、Svelte 5符文、SvelteKit路由） |
| Vue | `.vue` | 完全支持（脚本+脚本设置、Nuxt页面/API/中间件路由） |
| Liquid | `.liquid` | 全力支持 |
| Pascal / Delphi | `.pas`、`.dpr`、`.dpk`、`.lpr` | 完全支持（类、记录、接口、枚举、DFM/FMX表单） |
| Lua | `.lua` | 全面支持（函数、方法、本地、`require`导入、调用边缘） |
| Luau | `.luau` | 完全支持（ Lua ，加上类型签名， `type`别名， Roblox `require` ） |
